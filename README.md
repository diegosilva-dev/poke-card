# Pokemon Card

Projeto para realização de uma listagem de cards de pokemon por meio de uma API.


## Technologies
* [Flutter](https://flutter.dev)
* [API - pokemontcg.io](https://api.pokemontcg.io/v1/cards/)
* [Slidy](https://github.com/Flutterando/slidy)

### Main Packages
* Flutter Modular
* Mobx
* Dio
